Column settings component

Toggle column visibility and column order using the ColumnSettingsComponent.

Installation

Install with npm by running:

npm install @angular-generic-table/column-settings dragula ng2-dragula --save

More info available in the wiki.

Usage

This how the ColumnSettingsComponent looks and behaves by default but feel free to customize it using css or by passing predefined classes. You can also define your own template for the columns.

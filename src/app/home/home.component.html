Getting started

Angular Generic Table is a generic table component for Angular. It uses standard markup for tables ie. table, tr and td elements etc. and has support for expanding rows, row selection, global search, filters, sorting on multiple columns, pagination, export to CSV, column clicks, custom column rendering, custom export values, responsive layout etc.

Installation and usage

Install with npm by running:

npm install @angular-generic-table/core --save

More info available in the wiki, feel free to look at the basic setup too:)
